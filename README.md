# Airbuds Search Kakasi Adapter

This mod provides Kakasi-based romaji conversion for the [Airbuds Search mod](https://github.com/zilewang7/Beat-Saber-Airbuds-Search).

## Install
- Install this mod alongside `airbuds-search`.
- When loaded, `airbuds-search` will detect this adapter and use it for Japanese conversion.

## Notes
- This adapter is GPL-licensed because it links Kakasi.
- It bundles Kakasi dictionaries into:
  `/sdcard/ModData/com.beatgames.beatsaber/Mods/airbuds-search-kakasi/kakasi/`
